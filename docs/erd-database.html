Keterangan Notasi

PK Primary Key — identifikasi unik baris
FK Foreign Key — referensi ke tabel lain
UK Unique Key — nilai tidak boleh duplikat
NN Not Null — wajib diisi
TEXT?  Nullable — boleh kosong
1 → N  Relasi One-to-Many
